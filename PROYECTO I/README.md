**Sistema de Análisis y Gestión Empresarial** es una aplicación interactiva desarrollada en Streamlit que permite analizar información financiera, comercial y administrativa mediante diferentes módulos integrados. El proyecto aplica conceptos fundamentales de Python, análisis de datos y programación orientada a objetos en escenarios empresariales.

---

# 🚀 Introducción

Bienvenido a **Sistema de Análisis y Gestión Empresarial**, una plataforma diseñada para aplicar conocimientos de programación en Python en contextos prácticos relacionados con gestión financiera, análisis comercial y administración de información.

La aplicación integra funcionalidades de cálculo, visualización y gestión de registros mediante una interfaz dinámica e interactiva.

---

# 📈 Descripción del Proyecto

Este proyecto ofrece:

- **Análisis de flujo de caja** mediante registro de ingresos y gastos.
- **Gestión y análisis de productos** con cálculo de ventas totales.
- **Indicadores comerciales clave** para evaluar el desempeño del negocio.
- **Operaciones CRUD** utilizando programación orientada a objetos.
- **Visualización de información en tablas dinámicas**.
- **Interfaz interactiva e intuitiva** desarrollada en Streamlit.

---

# 🛠️ Requisitos

Para ejecutar la aplicación necesitas tener instalados los siguientes paquetes:

- Python 3.10 o superior
- Streamlit
- Pandas
- NumPy

Puedes instalar todos los requisitos usando el archivo `requirements.txt`:

```bash
pip install -r requirements.txt
